第二讲 启动和关闭浏览器
=======================

启动浏览器
----------

```ruby
Watir::Browser.new :chrome
```

关闭浏览器
----------

```ruby
b = Watir::Browser.new :chrome
b.close
#b.quit
```

命名空间
--------

Waitr是一个module，可以理解成命名空间。

命名空间简单来说可以理解成一个人的姓，同名的人很多，但是同名同姓相对就少一点。


类
--
Browser是一个类。

类可以理解成是图纸、模具、模板。

对象
----

```Waitr::Browser.new```产生了Browser类的一个对象。

如果类是图纸，那么对象可以理解成是根据图纸建造好的房子。
